Goal: Transform BlockVote from a hybrid blockchain/Web2 system into a pure Web2 application that eliminates all blockchain dependencies while maintaining the same election workflow.

Key Requirements:
No Blockchain Dependencies

Remove MetaMask/wallet requirements

Eliminate ETH transactions and gas fees

No smart contract interactions for voting

Pure Web2 Authentication

Username/password login only

No wallet connections required

Database-Driven Everything

All voting stored in MongoDB/MySQL

Phase management through Web2 API

Admin controls via database operations

Maintained Election Flow

Registration → Candidate Registration → Delegate Voting → Party Registration → Council Voting → Results

Same department restrictions and eligibility rules

Admin reset/revert capabilities at any phase

Simplified User Experience

Students just log in and vote

No crypto knowledge needed

Traditional web application feel
